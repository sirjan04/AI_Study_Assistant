# AI-Powered Learning Assistant System

An intelligent educational web application designed to improve student learning through chatbot-based interaction, quiz performance tracking, personalized study recommendations, and progress analytics.

The system uses a rule-based chatbot with Natural Language Processing (NLP) concepts such as keyword matching, regex patterns, and intent recognition to provide instant responses to student queries. It also includes a quiz module that tracks performance, identifies weak areas, and generates adaptive study recommendations.

The project is built using HTML, CSS, and JavaScript with a hybrid AI approach that combines rule-based intelligence and future-ready Generative AI integration. The architecture is prepared for advanced AI models like OpenAI GPT for future enhancements.

## How It Works

### 1. Chatbot Interaction

Users ask study-related questions through the chatbot interface.

### 2. Intent Recognition

The system processes input using keyword matching and regex patterns to identify user intent.

### 3. Response Generation

Predefined responses are delivered instantly based on topic classification.

### 4. Quiz Performance Tracking

Users attempt quizzes, and scores are recorded for progress analysis.

### 5. Personalized Recommendations

The system suggests study topics based on weak areas and quiz performance.

### 6. Analytics Dashboard

Users can monitor performance trends, progress metrics, and learning insights.
